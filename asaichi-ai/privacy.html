プライバシーポリシー

最終更新日: 2026年7月14日

本ポリシーは、朝イチAI（以下「本サービス」）のコンテンツ配信を自動化する 運用システム（以下「本システム」）における、情報の取り扱いについて定めるものです。 本システムは運営者個人が開発・運用しています。

1. 本システムの概要

本システムは、運営者が制作したAIニュースの解説動画を、 運営者自身が保有・管理する YouTube チャンネルにのみ 自動でアップロードするために、YouTube Data API v3 を利用します。

2. 利用する Google / YouTube の権限

  • https://www.googleapis.com/auth/youtube.upload
    運営者自身のチャンネルへ動画をアップロードする目的のみに使用します。 動画の閲覧・分析データの取得、他ユーザーのデータへのアクセスには使用しません。

本システムが利用する認証情報（OAuth トークン）は、運営者が管理する ローカル環境にのみ保存され、第三者に共有・送信されることはありません。

3. 取得しない情報

本システムは視聴者・第三者の個人情報を一切取得・収集・保存しません。 本システムが操作する対象は運営者自身のチャンネルに限られ、外部ユーザーとの データのやり取りは発生しません。Cookie やトラッキングによる利用者情報の 収集も行いません。

4. 情報の第三者提供

本システムは、いかなる情報も第三者に販売・提供・共有しません。

5. Google API 利用に関する遵守事項

本システムによる Google／YouTube API から取得した情報の利用および転送は、 Google API Services User Data Policy（限定的使用の要件を含む）を遵守します。

6. 改定

本ポリシーは、必要に応じて改定されることがあります。改定時は本ページに 最新の内容と更新日を掲載します。
